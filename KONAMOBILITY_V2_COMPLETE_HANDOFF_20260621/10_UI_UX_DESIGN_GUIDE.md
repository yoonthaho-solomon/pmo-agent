Principles:
- One core message per page.
- Head KPIs fixed and meaningful.
- Avoid repeated copy and card nesting.
- Use visual explanation before long text.
- Distinguish real data, simulation, and unsupported states.
- Buttons must be visibly clickable.
- Map and spatial stage should carry the main visual weight in Matching Studio.
